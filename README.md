# Hyderabad_house_price-prediction
This project aims to build a robust machine learning model to predict residential property prices in Hyderabad, one of India's fastest-growing metropolitan cities. With the real estate market booming and data-driven decisions becoming essential, this tool empowers buyers, sellers, and investors to make smarter choices.

Dataset: Cleaned and curated data from real estate listings in Hyderabad.

Modeling: Multiple regression models tested including Linear Regression, Decision Trees, Random Forest, and XGBoost.

Deployment: A user-friendly web interface built with Flask where users can input property details and get instant price predictions.
